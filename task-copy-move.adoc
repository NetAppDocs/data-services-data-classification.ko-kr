= NetApp Data Classification을 사용하여 파일을 복사, 이동 및 삭제합니다
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
거버넌스 및 비용 절감 기회에 대한 통찰력을 제공할 뿐만 아니라 NetApp Data Classification을 사용하면 Investigation 대시보드에서 파일에 대한 작업을 수행할 수 있습니다. Investigation 대시보드에서 데이터를 필터링한 후 비정형 데이터를 복사, 이동 및 삭제할 수 있습니다. 이러한 작업을 수행하면 파일을 더 안전한 위치로 이동하여 규정 준수를 개선하거나 오래되거나 중복된 파일을 삭제하여 비용 절감 기회를 창출할 수 있습니다.



== 파일 복사

Data Classification은 최대 1GB 크기의 파일 복사를 지원합니다. 총 1TB의 데이터 또는 500만 개의 파일 중 먼저 도달하는 제한까지 복사할 수 있습니다. CIFS 공유 또는 NFS 대상으로 파일을 복사할 수 있습니다. 파일을 복사할 때 소스 파일에는 영향을 미치지 않습니다.

.단계
. 조사 대시보드에서 복사할 파일을 선택합니다.
. 파일 목록 위에서 **복사**를 선택합니다.
+
image:screenshot-copy-move-delete.png["조사 페이지의 작업 스크린샷"]

. 복사본의 대상 프로토콜을 선택합니다. **NFS** 또는 **CIFS**.
. NFS를 선택한 경우, 슬래시를 사용하여 대상 경로를 입력합니다(예: `hostname:/path/to/destination`).
+
CIFS를 선택한 경우, 대상 경로를 백슬래시(\)를 사용하여 입력하십시오(예:  `\\server\share\path`). 또한 CIFS 공유에 대한 **Username** 및 **Password** 자격 증명을 입력하십시오.

+
image:screenshot-copy-nfs.png["NFS 파일 복사 기능의 스크린샷"]

. **제출**을 선택합니다.




== 파일 이동

Data Classification은 최대 1GB 크기의 파일 이동을 지원합니다. 총 1TB의 데이터 또는 500만 개의 파일 중 먼저 도달하는 제한까지 이동할 수 있습니다. CIFS 또는 NFS 대상으로 파일을 이동할 수 있습니다.

.단계
. 조사 대시보드에서 이동할 파일을 선택합니다.
+
image:screenshot-copy-move-delete.png["조사 페이지의 작업 스크린샷"]

. 파일 목록 위에서 **이동**을 선택합니다.
. 대상의 프로토콜을 선택합니다. **NFS** 또는 **CIFS**.
. NFS를 선택한 경우, 슬래시를 사용하여 대상 경로를 입력합니다(예: `hostname:/path/to/destination`).
+
CIFS를 선택한 경우, 대상 경로를 백슬래시(\)를 사용하여 입력하십시오(예:  `\\server\share\path`). 또한 CIFS 공유에 대한 **Username** 및 **Password** 자격 증명을 입력하십시오.

. 선택적으로 **Leave breadcrumb**를 선택하면 파일이 이동된 위치와 이유를 설명하는 breadcrumb 파일이 소스 볼륨에 남게 됩니다.
+
.. **Leave breadcrumb**을 선택한 경우 파일 이동 이유와 필요한 경우 새 위치를 설명하는 텍스트를 입력합니다.
+
image:screenshot-move-nfs.png["NFS 파일 이동 기능의 스크린샷"]



. **항목 이동**을 선택합니다.




== 파일 삭제

Investigation 대시보드에서 데이터를 필터링하여 스토리지 공간 및 비용 절감 기회를 찾을 때 해당 파일을 삭제할 수 있습니다. 파일 삭제는 비정형 파일에 대해서만 지원됩니다.

.단계
. 조사 대시보드에서 삭제할 파일을 선택하세요.
. 확인 대화 상자에서 "permanently delete"를 입력합니다.
. 작업을 확인하려면 **파일 삭제**를 선택하십시오.
+
image:screenshot-delete.png["삭제 대화 상자의 스크린샷"]
